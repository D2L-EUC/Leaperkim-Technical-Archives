# LeaperKim Lynx (Original) - Technical Documentation

The Lynx is LeaperKim's flagship lightweight performance wheel, known for its 151.2V system and hall-less motor control.

## ⚡ Technical Specifications

* **Voltage:** 151.2V Max (126V Nominal).
* **Battery:** 2700Wh (Samsung 50S 21700 cells).
* **Motor:** 3200W High Torque (Hall-less).
* **Controller:** 36-MOSFET design (200V rated).
* **Charging:** Dual GX20/5 pins (Max 16A total).

## 🛠️ Key Components & Specs

### Electronics
* **Mainboard:** Features 36x MOSFETs (TO-263 package).
* **BMS:** Smart BMS with individual cell monitoring via the display.
* **Display:** High-brightness LCD showing real-time amperage and voltage.
