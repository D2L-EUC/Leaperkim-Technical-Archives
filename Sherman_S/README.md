# LeaperKim Veteran Sherman-S - Technical Documentation

The Sherman-S (~2022) was the first suspension-equipped Veteran wheel, building on the original Sherman platform with the addition of FastAce suspension and an upgraded battery/motor.

## ⚡ Technical Specifications

| Parameter | Value |
| :--- | :--- |
| **Voltage** | 100.8V Max (84V Nominal) |
| **Battery** | ~3,500Wh (21700 cells, 24S configuration) |
| **Motor** | 2,800W Nominal |
| **Peak Power** | ~6,000W (estimated) |
| **Tire** | 20" × 2.5" (Tubeless) |
| **Top Speed** | ~45-50 mph (72-80 km/h) |
| **Range** | ~80-100 miles (depending on speed) |
| **Weight** | ~83 lbs (~38 kg) |
| **Max Load** | 330 lbs (150 kg) |
| **Suspension** | FastAce hydraulic, adjustable |
| **Charging** | Dual GX20 charging ports |
| **Frame** | Steel roll cage design (updated from OG) |

> ⚠️ **Note:** Some specifications above are estimates based on community data. Exact values need verification.

## 🛠️ Key Components

### Electronics
* **Controller:** Updated MOSFET design over the original Sherman.
* **BMS:** Improved BMS with cell monitoring.
* **Display:** Onboard LCD (speed, battery %, mileage, settings).
* **BLE Module:** UART Bluetooth for DarknessBot / EUC World.

### Mechanical
* **Suspension:** FastAce hydraulic shocks — the first suspension system in the Veteran lineup. Marked a major turning point for the brand.
* **Body:** Steel roll cage, revised from the OG Sherman to accommodate suspension linkage.
* **Pedals:** Long foot pedals.
* **Trolley Handle:** Retractable.

## 📝 Notes

* The Sherman-S introduced FastAce suspension to the Veteran brand — a partnership that continued through every subsequent model.
* Still uses 100.8V architecture (same as the original Sherman), making it the last Veteran model at this voltage before the jump to 151.2V.
* The "S" stands for **Suspension**.
* Shares many electronic components with the OG Sherman but with updated firmware.
