# LeaperKim Veteran Patton-S - Technical Documentation

The Patton-S (~2024) is Leaperkim's mid-size 17" suspension wheel, positioned as a more nimble and lighter alternative to the 20" Lynx. The "S" indicates it has suspension (similar naming convention to the Sherman-S).

## ⚡ Technical Specifications

| Parameter | Value |
| :--- | :--- |
| **Voltage** | 134.4V Max (112V Nominal) |
| **Battery** | 2,220Wh (Samsung 50S / 21700 cells) |
| **Motor** | 3,000W Nominal |
| **Tire** | 17" | 3.0-12 |
| **Top Speed** | ~50 mph (80 km/h) — *needs verification* |
| **Weight** | ~75 lbs (~34 kg) — *needs verification* |
| **Max Load** | 265 lbs (120 kg) — *needs verification* |
| **Suspension** | FastAce hydraulic |
| **Charging** | Dual charging ports |
| **Frame** | Magnesium alloy |

> ⚠️ **Note:** Several specifications above are estimates. Community verification needed.

## 🛠️ Key Components

### Electronics
* **Controller:** MOSFET-based controller (exact count TBD).
* **BMS:** Smart BMS with cell monitoring.
* **Display:** LCD display with speed, battery, and settings.
* **BLE Module:** UART Bluetooth for DarknessBot / EUC World.

### Mechanical
* **Suspension:** FastAce hydraulic shocks — carrying forward Leaperkim's suspension expertise.
* **Tire:** 17" × 3.0-12 — smaller and more nimble than the 20" Lynx class. Better suited for urban commuting and tighter trails.
* **Frame:** Magnesium alloy construction.

## 📝 Notes

* The Patton-S fills the mid-size niche in the Leaperkim lineup — between beginner wheels and the larger Sherman/Lynx/Oryx class.
* The 17" tire provides a more agile, responsive ride compared to 20"+ wheels, at the cost of some high-speed stability.
* Named after General George S. Patton — continuing Leaperkim's military naming theme (Sherman, Patton).
* Priced around $2,999–$3,499 USD (via eWheels as of 2025).
